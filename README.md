# 🐉 Projeto DBZ - Sistema de Gerenciamento de Guerreiros

Este projeto é uma **API REST Fullstack** desenvolvida para o gerenciamento de personagens. Ele demonstra a integração entre um back-end robusto em C# e um front-end dinâmico.

## 🚀 Tecnologias Utilizadas
- **Back-end:** .NET 8/9 (ASP.NET Core API)
- **Banco de Dados:** MySQL com Entity Framework Core (Migrations)
- **Front-end:** HTML5, CSS3 (Bootstrap 5) e JavaScript Assíncrono (Fetch API)
- **Versionamento:** Git e GitHub

## 🛠️ Funcionalidades Principais
- **CRUD Completo:** Cadastro, listagem e exclusão de guerreiros diretamente no banco de dados.
- **Validação de Dados:** Filtro no Front-end para evitar entradas vazias ou espaços desnecessários (`.trim()`).
- **Persistência Real:** Integração com banco de dados MySQL via XAMPP/Workbench.

## 📖 Como Rodar o Projeto
1. Clone o repositório.
2. Certifique-se de que o **MySQL** está rodando.
3. Ajuste a `Connection String` no arquivo `appsettings.json`.
4. Execute `dotnet watch run` no terminal.
5. Abra o `index.html` no navegador.
